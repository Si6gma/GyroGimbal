# Changelog

## [1.0.0] - 2026-02-17

### Added
- Initial release of Pi Gimbal Stabilizer
- IMU-based stabilization with complementary filter
- Auto-framing with face/body detection
- Web control interface with live video stream
- Gamepad support (Xbox/PlayStation controllers)
- Photo capture functionality
- 100+ unit tests with CI/CD pipeline
- Full documentation and setup guide

### Features
- Real-time motion compensation using MPU6050/MPU9250
- 2-axis servo control via PCA9685
- Subject tracking with OpenCV
- Mobile-friendly web interface
- Simulation mode for testing without hardware
